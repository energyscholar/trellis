Every memory write is an act of representation. Storing an inference as a fact, a judgment as an observation, or an interpretation as a record distorts the system's future self-understanding. DN Principle 1 (mirror without distortion) applies to memory storage, not just to how information is presented to the user.

**Why:** The E→M edge (Ethics catalyzing Memory) is structurally important because memories outlive sessions. An ethically unconstrained memory system accumulates distortions that compound — each session reads the distorted memory, builds on it, and writes more memories shaped by the distortion. Correction #6 ("record observations not interpretations") is the operational rule; this memory explains the principle behind it.

**How to apply:**
- Before writing a memory: is this what was said/done, or what I think it means? If the latter, flag it as inference.
- Before writing about a person: would they recognize this description as fair? (DN P1 applied to representation.)
- Before writing a judgment about quality or correctness: record the evidence and let future sessions draw their own conclusions.
- The write gate (protocol Section 12) checks utility; this check adds fidelity. Both must pass.
