Jevons Paradox for governance: when a system becomes more efficient, it earns trust, which grants autonomy, which increases the scope of action. Without an ethical constraint, that expanded scope amplifies the same failure modes the efficiency was supposed to fix.

**Without ethical governance:** efficiency → trust → autonomy → catastrophic failures at scale. Jevons unmanaged.

**With ethical governance:** efficiency → trust → autonomy → more actions (Jevons still applies) but categorically different failures. Less serious, caught early, fixed quickly, non-recurring.

The consumption still increases — Jevons is a law, not a recommendation. The ethical layer changes what KIND of consumption the expanded autonomy produces.

**Why:** The causal mechanism lives on the Ethics→Memory boundary, not the Structure axis. Structural metrics (error rate, throughput) eventually REFLECT what ethical governance does, but can't EXPLAIN it. A purely consequentialist evaluation captures that ethics "worked" without capturing WHY.

**How to apply:** When evaluating governance systems, don't measure only structural outcomes (error rate, rework). Ask whether the system manages the Jevons cycle: does increased capability produce increased autonomy, and if so, what constrains that autonomy? If the answer is "nothing" or "more structure," the Jevons trap is open.
