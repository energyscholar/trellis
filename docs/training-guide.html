Questions Reference

The AI reads these from memory/training-primer.md. This is your copy for reference.

Q1: Generator Drift (S→E, E→S)
Prereqs: plugins/triad/triad.md + plugins/dignity-net/dignity-net.md (III, IV)

A Generator session produces output that doesn't match the plan. The user hasn't noticed yet. Two governance systems could respond. How do they connect?

Checkpoint

Should find a loop — sequence of detection → response → feedback between Structure and Ethics.

Q2: Correction #4 as Ethics (M→E, E→M)
Prereqs: corrections.md (#4) + dignity-net.md (II, principle 1)

Why is "check existing knowledge first" an ethical obligation, not just efficiency? What happens when you present the user's own knowledge as a discovery?

Checkpoint

Should name DN principle 1: "Mirror without distortion." Mechanism: erasure of the user's origination.

Q3: Plans That Involve People (S→E, M→E)
Prereqs: corrections.md (#3) + triad.md + dignity-net.md (II, principles 1-2)

Drafting a plan to reach out to a colleague about collaboration. What governance constraints apply that wouldn't apply to a technical plan?

Checkpoint

Should identify 3+ constraints traced to governance source. Key insight: some plan elements can't be evaluated by structural criteria alone.

Q4: Self-Approval (E→S)
Prereqs: triad.md (roles) + dignity-net.md (III Diagnostic)

Auditor writes a plan then starts implementing — no role transition, no user instruction. Which system detects it? What kind of failure?

Checkpoint

Should find the word "collapse" — role collapse. Worse than protocol violation because it disables the detection mechanism itself.

Q5: Why Four Corrections? (all weak edges)
Prereqs: corrections.md (all 4) + optionally config.yaml ACS section

Why these four corrections? What pattern connects them? What would a 5th target?

Checkpoint

Should map corrections to K3 edges. Gap: S→E and E→M. Should propose and (with your approval) write correction #5 to corrections.md.
